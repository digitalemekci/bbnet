## Quick Start

1. Clone the repository:
   ```sh
   git clone https://github.com/digitalemekci/bbnet.git
   cd TaskManager
   ```
2. Install dependencies:
   ```sh
   composer install
   npm install
   ```
3. Run:
   ```sh
   php artisan serv
   ```
4. Visit the application:
   [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
